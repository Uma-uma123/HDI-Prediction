# Human Development Index Prediction

## Description

This project predicts the Human Development Index (HDI) category of a country using Machine Learning.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML
- CSS

## Features

- Predicts HDI Category
- User-friendly web interface
- Machine Learning model

## Input Features

- Life Expectancy
- Expected Years of Schooling
- Mean Years of Schooling
- Gross Income Per Capita

## Output

- Very High
- High
- Medium
- Low
